# Mert Acar - Mobile Back-End Görevleri ve Kanıtları

## Görev Dağılımı ve Branch Bilgileri

* **Grup Üyesi:** Mert Acar
* **Sorumlu Olduğu REST API Entegrasyonları:**
  * `GET /api/products` (Ürünleri Listeleme)
  * `POST /api/cart` (Sepete Ürün Ekleme)
  * `POST /api/orders` (Sipariş Oluşturma)
* **Çalışılan Branch:** `feature/mert-mobile-backend`
